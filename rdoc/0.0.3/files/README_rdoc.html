ms-core

The core mspire library for working with mass spectrometry proteomics data. Generally, This will be used as a dependency and won’t be all that useful on its own.

Description

  • Github
  • Lighthouse
  • Google Group

Installation

Available as a gem on RubyForge. Use: 

% gem install ms-mascot

Typically, it will be included as a dependency so it will be installed with another gem
